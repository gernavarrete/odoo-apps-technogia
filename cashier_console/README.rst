Cashier Console / Consola de Caja
================================

English
-------

The **Cashier Console** module provides a Point of Sale inspired experience to
manage cashier sessions in Odoo. Once installed it appears as its own
application tile and lets accounting and retail teams:

* Create cashier sessions linked to employees, recording opening and closing amounts.
* Monitor a dashboard with sales orders waiting to be invoiced and the ones already posted.
* Launch a payment wizard to generate invoices, register payments with the proper journal
  and open the resulting printouts in one flow.
* Configure from Settings which journals, taxes, permissions, cash rounding
  methods and currencies are available per company.
* Split cash collection journals between local and foreign currency while keeping
  access control centralized in Odoo.

Requirements
~~~~~~~~~~~~

The module depends on ``sale_management``, ``account``, ``hr`` and
``point_of_sale`` because it reuses employees, sales orders and accounting
journals to represent the collected payments.

Compatibility
~~~~~~~~~~~~~

The module is tested for **Odoo 18.0**.

Español
-------

El módulo **Cashier Console** ofrece una aplicación inspirada en Punto de Venta
para gestionar sesiones de caja en Odoo. Al instalarse aparece como una app
propia y permite a los equipos de finanzas y retail:

* Crear sesiones ligadas a empleados con registro de montos de apertura y cierre.
* Revisar un panel con las órdenes pendientes de facturar y las ya facturadas.
* Ejecutar un asistente que crea la factura, registra el pago con el diario
  correcto y abre las impresiones correspondientes en un solo flujo.
* Configurar desde Ajustes los diarios, impuestos, permisos, métodos de
  redondeo y divisas que cada compañía puede utilizar.
* Dividir los diarios de cobro entre moneda local y extranjera manteniendo el
  control de acceso centralizado en Odoo.

Requisitos
~~~~~~~~~~

Depende de ``sale_management``, ``account``, ``hr`` y ``point_of_sale`` porque
reutiliza empleados, órdenes de venta y diarios contables para reflejar los
cobros registrados en caja.

Compatibilidad
~~~~~~~~~~~~~~

La versión del módulo está probada para **Odoo 18.0**.
