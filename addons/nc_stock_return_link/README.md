NC Stock Return Link
====================

Automatically links credit notes to stock returns and prevents editing of lines with associated returns.

Installation
------------
1. Install the module normally through Odoo Apps
2. The functionality works automatically when creating credit notes from invoices

Configuration
-------------
No configuration needed. Works out of the box.

Usage
-----
1. Go to Accounting → Customer Invoices
2. Select an invoice with stock returns
3. Click "Reverse" to create credit note
4. System automatically adjusts quantities based on returns
